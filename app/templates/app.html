Convert transactions CSV files

Transform a DEGIRO transactions CSV file to a CSV format suitable for importing into a Simply WallSt transactions portfolio.

Usage

  1. Select the transactions CSV file exported from DEGIRO.
  2. Select the currency used by the destination Simply WallSt portfolio. Other currencies are filtered out.
  3. Clicking convert will prompt with a CSV-file to download. This file can be used in Simply WallSt to create or update a portfolio.
