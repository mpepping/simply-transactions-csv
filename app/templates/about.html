About

This app can be used to convert a DEGIRO transactions CSV file to CSV format suitable for importing into a Simply WallSt portfolio.

How do I use it?

See usage on the landing page.

  • DEGIRO provides information on exporting transactions to a CSV file.
  • Simply WallSt provides information on importing CSV-files in a Portfolio.

Data privacy

This app does not collect or store any personal data. No transaction data is collected or stored, in any form. The only information stored is a usage count of this app, using request containing the client IP-address and corresponding date/timestamps.

If you don't want to use this service, but instead run your own instance of this service, or just see the source code: This app is open source and can be found on GitHub.

Disclaimer

This app is provided as is. No guarantees are made that the output is correct. Use at your own risk.

Why an app for an easy thing as a CSV transformation?

The DEGIRO transactions CSV file contains all transactions over a period of time but differs strongly from the CSV file format that Simply WallSt. uses. Since converting formats requires some lookups and transformations, I found it more useful to create an app to do this in a reproducible way.

More in detail, the DEGIRO file contains 19 columns with data where the Simply WallSt. CSV file expects 5 columns. Out of these 5 columns all matching columns from the source CSV file require some sort or transformation before the data is usable for import in Simply WallSt., varying from:

  • Converting the date format
  • Lookup ISIN numbers to Ticker symbols
  • Filtering based on currency
  • Inverting negative stock prices
  • Adding a Buy/Sell action field
